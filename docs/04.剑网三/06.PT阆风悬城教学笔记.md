# 25人普通阆风悬城 教学笔记

> 副本位置：昆仑 · 阆风悬城
> 首领数量：5（笑妆娘 → 唐醉 → 柳公子 → 阿史那承庆 → 唐怀仁）
> 狂暴时间：各BOSS均为8分钟
> 难度定位：全民PVE，机制比弓月城友好，前三个偏木桩，后两个吃配合

---

## 一、团队招募 & 配置要求

### 推荐阵容

| 配置方案 | T | 奶 | DPS | 说明 |
|----------|----|-----|------|------|
| 舒适团 | 3-4 | 5-6 | 15-17 | 推荐教学团使用，容错高 |
| 标准团 | 3 | 5 | 17 | 常规farm配置 |
| 极限团 | 2 | 4 | 19 | 需要全员熟练，不推荐教学 |

### 职业预留

**T（坦克）必留：**
- 老二唐醉「碎星·刚」需要 **2个T各踩一个特殊圈**（非T踩即死），至少保证2T
- 老三柳公子平A叠buff需换T（6层一换），建议3T轮换
- 老四阿史那承庆「刑」debuff需副T换仇，且「处决」需副T扶摇承伤
- 推荐配置：**主T（明教/少林/天策）+ 副T（天策/苍云）+ 备用T**，其中天策T处理老三三叠扇有奇效

**奶妈必须够：**
- 老二「毒变」需奶满再驱散，驱散后持续掉血
- 老四「鬼首」「巨刃掠影」全团高压抬血
- 老五「磁震」「玄阴霹雳弹」对奶量要求高
- 推荐：**至少5奶，带1-2个能驱散的奶**（如奶花/奶毒/药奶）

**远程DPS必备：**
- 老二「魂锁牵」点名远程拉断连线（点8远程含奶）
- 老五P1「索命箭」点名3远程D跑圈
- 建议带 **6-8个远程DPS**，不足则奶妈凑

**近战DPS注意事项：**
- 老四「魂火唤醒」近战先消，需要近战去踩魂火
- 老五P2多个技能近战需频繁走位

### 招募公告模板

```

T补2K 奶补1K DPS检查装备
要求：装分22W+ 附魔齐全 看过攻略
来不玻璃心 听指挥 犯错罚款制
================================
预留：主T/副T/奶花/奶毒 各1
```

## 二、全局通用机制 & 设置提醒

### 特效屏蔽警告

老四阿史那承庆的火球、老五唐怀仁的部分技能特效**不能通过 `Ctrl+Shift+N/P` 屏蔽**，否则看不到关键技能预警。

**解决方案：**
- 通过「效果性能设置」按技能单独设置屏蔽
- 或干脆不屏蔽特效（推荐指挥用）

### 标点导入方法

在游戏内打开团队标记面板 → 导入 → 粘贴下方JSON代码即可。

## 三、老一 · 笑妆娘

### BOSS概述

- **类型**：时间轴BOSS，纯木桩
- **场地**：正方形九宫格，有9根带空气墙的柱子
- **关键**：BOSS血量 80% / 50% / 10% 时会换位置并降下对应柱子（柱子空气墙消失，可站人）

### 技能循环

**固定循环：灵簪锁魂 → 祭跪双坟 → 悲恸音潮 → 循环**

血量轴换位时会等当前技能放完再移动。

### 技能详解

#### ① 灵簪锁魂

| 项目 | 说明 |
|------|------|
| 机制 | 随机点名1人，5秒后朝该玩家方向释放 **25尺90°扇形AOE** |
| 残留 | 扇形区域残留5秒地毯伤害，踩上去每0.5秒叠1层「精神匮乏」（减疗） |
| 处理 | 被点名者 **立刻去侧面/背后排扇形**，不要把扇形对着大团 |

#### ② 祭跪双坟

| 项目 | 说明 |
|------|------|
| 机制 | BOSS移到柱子间隙，在九宫格 **两横两竖** 四条通道成对刷新矩形AOE |
| 顺序 | **先竖后横**，矩形填充完之前走出范围 |
| 伤害 | 被打中叠1层「血染」（10秒）+ 4层「精神匮乏」 |
| 处理 | 看清楚是竖还是横先出，**走位躲避就行，不需要交位移** |

#### ③ 悲恸音潮

| 项目 | 说明 |
|------|------|
| 机制 | BOSS跳到未降落的柱子旁，8秒读条后扩散音波环 |
| 范围 | 最大扩散30尺，接触其他柱子会连锁扩散 |
| 关键 | **音波环高度 < 二段跳高度**，跳起来即可规避 |
| 处理 | 看到读条就挂好扶摇或直接二段跳，**伤害比特效先到，提前跳** |

### 血量轴换位

| 血量 | BOSS移动 | 降下柱子 |
|------|----------|----------|
| 80% | 1号点 → 2号点 | 一、二号台子 |
| 50% | 2号点 → 3号点 | 四、五、七号台子 |
| 10% | 3号点 → 4号点 | 三、六、八、九号台子 |

### 灭团点排查

- **悲恸音潮死亡**：站在多个音波环交接区没跳，或者跳晚了
- **灵簪/祭跪死亡**：单技能打不死人，死了基本是呆卡萌或海外高延迟 → 建议劝退
- 老一打不过的基本不用指望后面，尤其是老四老五

## 四、老二 · 唐醉

### BOSS概述

- **场地**：圆形，分 **风区**（半径15尺内，青铜网地面）和 **边区**（15-30尺环形）
- **场地边缘**：走出去会掉落，二段跳可跳回
- **关键**：有血量下P机制

### 技能详解

#### ① 透骨刺（承伤）

| 项目 | 说明 |
|------|------|
| 机制 | 点名玩家需要承伤，**最低5人进圈分摊** |
| 注意事项 | 承伤后会获得debuff，**debuff消失前不可再次进入承伤圈** |
| 处理 | 第一波5人进，第二波换另外5人，轮流承伤 |

#### ② 魂锁牵（拉断连线）

| 项目 | 说明 |
|------|------|
| 机制 | 点名 **远程玩家**（含奶妈），两两连线 |
| 处理 | 连线的两人 **分别反方向聂云40尺拉断** |
| 路线 | **环绕中场聂云，不要走中场风区** |

#### ③ 巨大风扇（击退）

| 项目 | 说明 |
|------|------|
| 机制 | 全场范围击退，吹向场地边缘 |
| 处理 | 开减伤/免控/抵抗技能，**别被吹出场外** |

#### ④ 三叠杀（打T）

| 项目 | 说明 |
|------|------|
| 机制 | 对当前T的连续高伤技能 |
| 处理 | T开减伤即可，奶妈注意看好T血 |

#### ⑤ 碎星（踩圈）+ 毒变（驱散）

这是老二的核心组合机制，流程如下：

**碎星阶段：**
- 远程 **散开踩圈**，每个圈都必须有人踩，**不可漏踩**
- 其中有 **两个特效圈只能T踩**，非T职业踩之即死
- 2个T分别负责各踩一个

**毒变阶段（碎星后立刻）：**
- 远程 **留在中场铁丝网区域**等待点名
- BOSS点名 **2人获得「毒变」debuff**
- 被点名的2人 **分开站，但只能在中场范围内**
- 没被点名的远程 **立刻聂云出中场**（到边区）
- 奶妈：**先奶满 → 再驱散**（顺序不能反！）
- 被驱散的玩家：驱散后立刻聂云出毒圈

### 灭团点排查

- 碎星漏圈 → 全团高伤
- 毒变驱散时机不对（没奶满就驱）→ 暴毙
- 魂锁牵没拉断/走风区 → 被定身/秒杀

## 五、老三 · 柳公子

### BOSS概述

- **核心机制**：扇子追击、换T、掠影探囊（金币检测）
- **起手2T抱团**，叠buff时6层一换，不叠时2T抱团防击退

### 技能详解

#### ① 公子扇·裂风（溜扇子）

| 项目 | 说明 |
|------|------|
| 机制 | 随机点名1人，产生一把追击扇子 |
| 金色阶段 | 扇子速度较快，**不可被追上**，被点名者聂云去场地边角 |
| 白色阶段 | 扇子变白后速度减慢，**回头与扇子碰撞引爆** |
| 其他人 | 躲好扇子残留的光点路径伤害 |
| 替补 | 原地会出第二个扇子，**替补队员去引导** |

#### ② 公子扇·聚锋（排矩形）

| 项目 | 说明 |
|------|------|
| 机制 | 点名玩家释放矩形AOE |
| 处理 | 去大团 **侧面** 排矩形，别对着人群 |

#### ③ 妙手空空（金币检测）

| 项目 | 说明 |
|------|------|
| 机制 | 检测携带金币数量 |
| 处理 | 身上 **没2砖（2Z）的留大团**承伤，**有钱的（≥2Z）去侧面排** |

#### ④ 三叠扇（核心难点）

| 项目 | 说明 |
|------|------|
| 技能序列 | **扇 → 剑 → 扇** 或 **剑 → 扇 → 剑** |
| 扇处理 | 被点名者去 **边角** 排 |
| 剑处理 | 被点名者去 **侧面** 排 |
| 替补 | 第二个扇子刷新后 **替补去接** |

**策T单人处理方案（推荐）：**
1. 1号点名扇让被点名者去左前角落
2. 3号点名扇让被点名者去右前角落
3. 策T贴近1号扇方向，接分裂后的新扇子
4. 带着所有扇子缓慢移动 → 等所有扇子变白
5. 开外功减伤 → **扶摇跳最高点滞空** → 高空引爆（>4尺规避扇骨伤害）
6. 引爆后聂云远离残留弹幕

### 灭团点排查

- 扇子没溜好/撞早了 → 提前引爆炸团
- 三叠扇替补没到位 → 扇子进人群
- T换嘲慢了 → 倒T

## 六、老四 · 阿史那承庆

### BOSS概述

- **场地**：九宫格，除中心外全部被 **魂火** 覆盖
- **魂火伤害**：进入魂火区域受穿透阴性内功伤害 + 2层「耐力损耗」
- **魂火唤醒**：可通过机制消除魂火

### 技能详解

#### ① 魂火唤醒

| 项目 | 说明 |
|------|------|
| 机制 | 点名2人，去踩魂火消除 |
| 顺序 | **近战先踩** → 远程后踩 |
| 处理 | 被点名者按顺序去踩魂火格子，消除对应区域魂火 |

#### ② 处决（承伤）

| 项目 | 说明 |
|------|------|
| 机制 | BOSS点名主T释放死刑级伤害 |
| 处理 | **副T扶摇跳起承伤** |
| 注意 | 扶摇最高点无法承伤，要在上升/下降途中接 |

#### ③ 归尘

| 项目 | 说明 |
|------|------|
| 机制 | 点名3人，箭头标记 |
| 处理 | 找安全点站位，**箭头朝外**（不要对着人群） |

#### ④ 斩 & 破

| 技能 | 说明 |
|------|------|
| **斩** | 黄色预警跳 → **红色预警不跳**，共闪5次，看清楚黄色是第几次 |
| **破** | 红色矩形预警，**找安全格子站位**，必要时交位移快速到位 |

#### ⑤ 巨刃掠影

| 项目 | 说明 |
|------|------|
| 机制 | BOSS回中场释放巨额AOE |
| 处理 | 全团回中场 → 奶妈全力刷血 |
| T处理 | 非主T离开BOSS正面180°，副T在6层debuff时去正面换仇，主T走 |

#### ⑥ 黄泉斩

| 项目 | 说明 |
|------|------|
| 机制 | **三连扇形 + 普攻死刑三连** |
| 处理 | 扇形左右躲，全员远离主T正面180° |
| 换T | **副T在第三次死刑时接仇** |

#### ⑦ 风暴连击

| 项目 | 说明 |
|------|------|
| 机制 | T进入黄泉领域，根据填充速度按QTE |
| 处理 | 带QTE的T进去按，其他T预备 |

#### ⑧ 鬼首

| 项目 | 说明 |
|------|------|
| 机制 | 全场鬼首弹幕 |
| 处理 | 奶妈群刷血，全员躲鬼首 |

### 灭团点排查

- 魂火没消够 → 场地不够用 → 后续技能没地方躲
- 黄泉斩换T慢了 → 主T暴毙
- 巨刃掠影没回中场/奶没刷起来 → 团灭

## 七、老五 · 唐怀仁

### P1 阶段（机甲战）

**核心机制**：场中有「机甲」可与之换位。

#### P1 技能流程

| 顺序 | 技能 | 处理方式 |
|------|------|----------|
| 1 | **开场** | 在场中放下机甲 |
| 2 | **磁震** | 全团AOE，奶妈刷血 |
| 3 | **索命箭** | 点名3个远程DPS聂云出人群，越远越好；脚下出红圈则换位，换完重新放机甲 |

#### P2 关键机制详解

**连弩扫射（灭团高发区）：**
1. 听到DBM提示立刻注意扫射方向
2. **2秒后点机甲换位**（不要提前换）
3. 换位后跟着扫射方向同向跑
4. 扫射结束 → BOSS脚下重新放机甲

**飞镖机关·切割：**
- 挂好扶摇，**提前一点起跳**（别等特效出来再跳）
- 空中保持滞空直到飞镖结束

**玄阴霹雳弹：**
- 主T注意：必须奶满再走，倒计时结束前开减伤
- 其他人：远离主T 15尺外

### P3 阶段（斩杀）

P2在 **5分钟内** 打下P3，否则狂暴。

P3血量计算公式：
```
P3血量 = 20% + (P2剩余血量 - 70%)
（如果P2剩余血量 - 70% < 0，则按0计算）
```

**例子：**
- P2结束时BOSS剩75% → P3血量 = 20 + (75-70) = **25%**
- P2结束时BOSS剩60% → P3血量 = 20 + 0 = **20%**

P3基本是**打木桩**，没有复杂机制，全力rush即可。

### 唐怀仁灭团点排查

- **连弩扫射**：换位时机不对/跑反方向 → 秒杀
- **飞镖切割**：跳晚了 → 被击飞秒杀
- **玄阴霹雳弹**：主T没开减伤/没远离 → 炸团
- **落星炙炎**：踩残留 → 80%减疗奶不上 → 慢性死亡

## 八、指挥 checklist

### 开打前

- [ ] 检查团队配置：T ≥ 2，奶 ≥ 4，远程D ≥ 6
- [ ] 确认有驱散职业（老二毒变）

## 九、团队公告模板（游戏内复制用）

### 笑妆娘
```
灵簪锁魂：点名去侧面排扇形，残留别踩
祭跪双坟：横竖两次交叉矩形，先竖后横，走位躲
悲音恸潮：跳环！伤害比特效先到，提前二段跳
```

### 唐醉
```
透骨刺：点名承伤最低5人，有debuff别再进
魂锁牵：远程连线反方向聂云40尺拉断，环中场走
碎星：远程散开踩圈别漏，2个特效圈只能T踩
毒变：留中场等点名，奶满再驱，驱完聂云出
```

### 柳公子
```
裂风：溜扇子去边角，金色别被追，白色回头撞
聚锋：去大团侧面排矩形
掠影：没钱(<2Z)留大团，有钱去侧面
三叠扇：扇去边角剑去侧面，替补接2扇
起手2T抱团，6层换T
```

### 阿史那承庆
```
魂火唤醒：点2人，近战先消远程后去
处决：副T扶摇承伤
归尘：找安全点，箭头朝外
斩：黄跳红不跳，闪5次
巨刃掠影：回中场奶血，副T 6层换仇
黄泉斩：三连扇形+死刑，副T第三次接仇
```

### 唐怀仁 P1
```
场中放机甲
索命箭：点3远程聂云出，脚底红圈就换位
充能余波：光幕扩散，和机甲换位
毒刹：2号圈放机甲→聂云去1号圈→换位
连弩：T面向朝外，其他人远离正面
```

### 唐怀仁 P2
```
朝BOSS走几步放机甲
玄阴霹雳弹：奶满主T，倒计时结束主T开减伤远离15尺
落星炙炎：半场AOE躲，别踩残留(减疗80%)
连弩扫射：2秒后换位→跟着扫射方向跑
飞镖切割：挂扶摇提前跳！
```
