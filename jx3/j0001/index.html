# 剑三相关网站合集

提示

个人收集使用，仅供参考

魔盒(JX3Box)

描述：剑三攻略网站，玩家运营官方合作，主要提供门派玩法攻略、游戏内信息、玩家分享攻略
链接：https://www.jx3box.com/index/ (opens new window)

隐元秘鉴

描述：剑三成就攻略网站，玩家运营，主要提供各成就攻略、玩家分享
链接：http://jx3yymj.com/ (opens new window)

剑网3开放预览

描述：剑三捏脸分线网站，玩家运营，提供捏脸真实预览
链接：https://faces.jx3openplayer.com/lib (opens new window)

JX3API

描述：剑三数据查询接口，常作为剑三机器人主要接口
链接：https://www.jx3api.com (opens new window)

剑网3·科举查询

描述：剑三科举答题查询助手，茗伊插件提供
链接：https://j3cx.com/exam/ (opens new window)

宏库-剑三官方宏 历史的眼泪!

描述：早年间PVE按键宏分享、获取的主要网站，现已升级为魔盒
链接：http://www.jx3pve.com/forum-294-1.html (opens new window)

剑网3·配装器 历史的眼泪!

描述：早年间PVE装备配装常用网站，现已停止维护
链接：https://www.j3pz.com/ (opens new window)
