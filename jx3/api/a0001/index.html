# 单配方成本查询

请求说明

查询指定配方指定数量制作所需成本及交易行价格

请求方式(POST)

http://costing.hidewnd.cn/costing/one
1

请求参数

{
  "server": "剑胆琴心",
  "formulaName": "风语·二十四桥明月夜",
  "number": 1,
  "rangeCreate": true
}
1
2
3
4
5
6
[√] 可选的[server]指定区服，查找该区服的相关记录；
[√] 必选的[formulaName]技艺配方名称；
[√] 可选的[number]需要制作的数量
[√] 可选的[rangeCreate]产出数量是否随机，为否则采用最小产出数

# 多配方成本查询

请求说明

查询多个配方指定数量制作总计所需成本及对应交易行总价

请求方式(POST)

http://costing.hidewnd.cn/costing/list
1

请求参数

{
  "server": "",
  "rangeCreate": true,
  "items": [
    {
      "formulaName": "风语·太后饼",
      "number": 5
    },
    {
      "formulaName": "风语·三鲜粥",
      "number": 5
    }
  ]
}

[√] 必选的[formulaName]技艺配方名称；
[√] 可选的[number]需要制作的数量
