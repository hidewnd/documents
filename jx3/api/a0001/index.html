# 单配方成本查询

请求说明

查询指定配方指定数量制作所需成本及交易行价格

请求方式(POST)

http://costing.hidewnd.cn/costing/one
1

请求参数

{
  "server": "剑胆琴心",
  "formulaName": "[断浪·腕·绣（外破）]",
  "number": 2
}
1
2
3
4
5

[√] 必选的[formulaName]技艺配方名称；
[×] 可选的[number]需要制作的数量，为空默认为1

1

{
  "server": "剑胆琴心",
  "items": [

      "formulaName": "断浪·上品破秽散",
      "number": 10
    },
