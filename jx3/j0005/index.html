PT弓月城教学笔记

# 老一 巴图

2T 四奶 19DPS

每块平台承伤三次后破裂 排圈不要擦边

金刚伏魔震

点名 两个远程， 被点名分别去台子边上

金怒明王杵

点名一个T 靠边 倒计时2秒时往外折返跳

六字大明咒

三波按顺时针防线的双大矩形

第一波时结算前走到安全点

第二波结算前往右手边走

第三波结算前往右手边聂云

如果下一个安全点已经裂了，则聂云去下一个的对角位置

镇魂梵音劫

三次AOE

# 老二 竭勒

苍焰流虹

随机点名 一个 排火圈

场地随机刷新一个火圈

被点名的出人群 15尺以外

离弦

5条细矩形

自行躲避

苍原孤月

连续三次左右大扇形

BOSS抱团左右躲

破阵斩

BOSS无敌，连续三次黄矩形

三个T轮流站线上承伤

# 图南子

普攻 叠加 debuff 北冥寒劲，可驱散

击浪式

大矩形+ 海浪

二段跳/扶摇规避

跃海式

箭头方向 大红圈，红圈结束扩散跳环

读条结束前 T引导BOSS往箭头圈边排大圈，

25普通模式下技能循环为：击浪式 → 跃海式 → 击浪式 两次击浪式面向相反

摘星

点名三个DPS拉到天上，二段跳或后跳规避/上天后下拉二段跳别摔死

荡波式

读条结束 全团AOE ，30sdot伤害

奶妈奶团血

# 老四 叶葵

固定狂暴： 5分50秒

2T 换仇恨

坠刃

读条3秒后在场地4个随机位置落下匕首

匕首周围4尺范围生成技能特效圈， 携带debuff【链命】玩家进圈必死

锁影缠身

读条3秒点名当前主T (debuff【锁影缠身】) ，读条结束 T中心十尺aoe

T聂云出人群保持10尺之外， aoe结束后换仇

链命

读条3秒后点名3组人

  • 【链命】（沉默、减速10%，20秒）
  • 【链命崩解】（100层，组内BUFF同步）

连线中点固定为匕首的中心，当两人保持在刀背方向、连线呈直角或锐角且保持移动时，debuff【链命崩解】层数减低、跳伤害

两人一左一右从刀刃侧向刀背侧移动，连线呈直角关系后左右或前后磨蹭几下即可解掉DEBUFF

蓝头顶 重要buff列表从上到下 1、2、3个蓝色人负责找圈 ，红找蓝

刀刃到两个的线段总长度小于15尺

常见处理方式：被点名玩家 插件提示蓝色头顶找同组红色头顶， 头顶找刀圈

索命旋刃

刷新4把旋转飞刀，并点名离飞刀最近的人跟随， 飞刀两两互碰消失

安排3N1T接近飞刀引导飞刀处理

裂空旋刃杀

读条5秒后倒读条以1秒为间隔释放5次外功穿透AOE伤害

奶妈抬血

# 老五 无面鬼

T 八层换仇

飘摇伞击

平A，给当前T附带一层debuff【伞击】，另外BOSS的平A在向9层debuff【伞击】玩家释放时会变成【跃潮斩波】，造成秒杀效果，也就是不能叠到10层

破阵曲·徵

读条后对所有玩家释放debuff【破阵曲·徵】封轻功，同时在场地中央放置一个大球，随后出一圈小球向周围扩散。其中8个球会随机在女神像方向或门口方向呈直角分布（第一轮）或呈平行分布（第二轮）或每条边两个（第三轮）固定位置停止并变成角圈。碰到小球的玩家会被叠加debuff【脆弱】：增加受到的伤害。

破阵曲·角

读条2秒，间隔3秒分两次点名，

每次点4个人并挂debuff【破阵曲·角】，

3秒后【破阵曲·角】变为 debuff【破阵曲·角·爆发】并开始跳伤害，5秒后debuff【破阵曲·角·爆发】结算并在玩家当前位置落圈（后文简称为“角圈”）

若圈有重叠，则触发角圈的惩罚机制：消去两个圈中的一个，每消一个圈对全团释放【破阵曲·角·共鸣】并给受到伤害的玩家叠加4层debuff【脆弱】

巽风掌

读条3秒并延迟1.5秒，以面向-背向的顺序连续两次180°扇形，命中会被推。

震雷引

BOSSS跳回场地中间，读条3秒后点名两个玩家，7秒后以两人为中心按点名提示的方向释放十字花射线伤害，射线伤害碰到角圈会激发出【破阵曲·徵】的小球，并无任何惩罚机制的消去角圈。碰到小球的玩家会被叠加debuff【脆弱】。

艮山掌·二式

读条10秒后击退所有玩家。若场地中存在角圈则触发上文中角圈的惩罚机制

定波砥澜

BOSS跳回场中间，点名一个玩家同时读条6秒后释放20尺圈并持续拉人8秒并反伤，结束后留在圈内的人会被秒杀；BOSS在被点名玩家倒计时结束后向该玩家当前位置释放一道飓风，到达位置后变为大风圈（伤害高度为二段跳高度），进风圈会被秒。

击水三千

T引导三段技能，

一式为读条3秒后释放90°扇形面向，

二式为读条2秒后释放半圆面向，

三段为读条1秒后释放矩形。每层会给被命中的玩家附带10层debuff【脆弱】。

浮游天地

BOSS跳会场中间，点名四个人获得debuff【伞影】（5秒）同时开始读条5秒，读条结束后BOSS无敌并将被点名玩家带上天，4秒后释放【逐波灵游】将被点名玩家击落到其升空位置约5尺后的地面，落点处会出现黄圈，1秒后根据黄圈内人数结算伤害并击倒圈内玩家、落下小风（附带减速效果），随后几秒从BOSS位置落下分身对小风内释放【跃潮斩波】秒杀小风内玩家并消除小风。

若黄圈结算时对圈内玩家造成致死伤害，则触发惩罚机制：BOSS对全团释放【逐波灵游·爆】并叠加debuff【易损】（持续到脱战）。

玩家被击倒后需要奶妈奶满解除击倒。

跃潮斩波

点名T心法玩家并挂上debuff，倒计时结束后落圈并在几秒后对圈内释放致死伤害，

若圆心在大风内，则大风分裂成数个飓风并向场边扩散。结束后若大风不存在，则BOSS获得易伤【虚弱】（受到伤害提高30%，20秒）；

若大风存在，则全场玩家叠加debuff【易损】（持续到脱战）
